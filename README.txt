ESTE PROJETO CONSISTE NA PARTE BACK-END DO PROJETO DE "GESTÃO DE AERONAVES"

-- É um projeto com arquitetura RESTFul onde existe os endpoints para o CRUD de aeronaves até consultas mais específicas
-- Para executar o projeto é necessário instalar o Java 8, uma IDE de sua preferência(Intellij, Eclipse, etc) e configurar as variáveis de ambiente '%JAVA_HOME%'
-- Após abrir o projeto  e baixar sua dependências, é possilvel iniciar o projeto
-- Após iniciar o projeto é possível ter acesso aos endpoints pela URL http://localhost:8080/ENDPOINT
